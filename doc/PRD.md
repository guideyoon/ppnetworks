# 피플네트웍스 홈페이지 PRD

## 1. 프로젝트 개요

### 프로젝트명
피플네트웍스 공식 홈페이지

### 목적
피플네트웍스는 소상공인을 대상으로 홈페이지 제작을 전문으로 하는 서비스 브랜드입니다. 본 홈페이지는 서비스 신뢰도 확보, 문의 전환 극대화, 제작 사례 홍보를 핵심 목표로 합니다.

### 핵심 목표
- 소상공인 대상 서비스 이해도 향상
- 홈페이지 제작 문의 전환율 극대화
- 포트폴리오 및 실제 사례 중심의 신뢰 구축
- 유지보수가 쉬운 구조로 장기 운영 가능

### 주요 타겟
- 소상공인, 자영업자
- 1인 사업자
- 오프라인 매장 운영자
- 홈페이지 제작 경험이 없는 사용자

---

## 2. 주요 기능 정의

### 2.1 공통 기능
- 반응형 웹 지원 (모바일, 태블릿, PC)
- SEO 기본 최적화
- 빠른 로딩 속도
- 접근성 고려한 UI

### 2.2 핵심 기능 목록

#### 1) 메인 페이지
- 서비스 핵심 메시지 전달
- 강력한 CTA (문의하기)
- 제작 프로세스 요약
- 대표 포트폴리오 노출

#### 2) 서비스 소개
- 홈페이지 제작 상품 설명
- 가격 정책 안내
- 제작 범위 및 포함 사항 설명

#### 3) 포트폴리오
- 업종별 제작 사례 리스트
- 상세 페이지 제공
- 실제 운영 사이트 링크

#### 4) 문의하기
- 간편 문의 폼
- 전화, 카카오톡, 이메일 연동
- 문의 완료 후 안내 메시지 제공

#### 5) 회사 소개
- 브랜드 스토리
- 작업 방식 및 철학
- 연락처 및 위치

#### 6) 관리자 기능 (향후 확장)
- 포트폴리오 등록/수정
- 문의 내역 조회
