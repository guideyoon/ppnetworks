# 서버 실행 가이드

## 중요: 서버 실행 방법

에러 코드 -102는 서버에 연결할 수 없다는 의미입니다. 서버가 실행되지 않았거나 접근할 수 없는 상태입니다.

### ✅ 올바른 실행 방법

**1단계: 터미널(CMD 또는 PowerShell) 열기**

**2단계: 프로젝트 폴더로 이동**
```cmd
cd "C:\Users\user\홈페이지\ppnetworks"
```

**3단계: 개발 서버 실행**
```cmd
npm run dev
```

**4단계: 서버가 시작될 때까지 대기**
다음 메시지가 나타날 때까지 기다리세요:
```
▲ Next.js 16.1.1
- Local:        http://localhost:3004
- Ready in X.Xs
```

**5단계: 브라우저에서 접속**
- `http://localhost:3004` 입력

### ⚠️ 주의사항

1. **터미널 창을 닫지 마세요**
   - 서버를 실행 중인 터미널 창을 닫으면 서버가 종료됩니다.

2. **포트 3004가 사용 중인 경우**
   - 다른 터미널에서 이미 서버가 실행 중일 수 있습니다.
   - `netstat -ano | findstr :3004` 명령어로 확인

3. **서버를 중지하려면**
   - 터미널에서 `Ctrl + C` 누르기

### 🔧 문제 해결

#### 포트가 이미 사용 중인 경우
다른 포트를 사용하세요:
```cmd
npx next dev -p 3005
```

#### 의존성이 설치되지 않은 경우
```cmd
npm install
```

#### 캐시 문제인 경우
```cmd
rm -rf .next
npm run dev
```

### 📝 확인 사항

서버가 정상적으로 실행되면:
- ✅ 터미널에 "Ready" 메시지 표시
- ✅ `http://localhost:3004`로 접속 가능
- ✅ 홈페이지가 정상적으로 표시됨
